# frontend

React 기반 프론트엔드

## 기술 스택
- React 
- css module
- Redux Toolkit
- Axios
- REST API 연동

## 주요 기능
- 메인 페이지 (축제 목록 슬라이드/문화영상)
- 축제행사 페이지 (목록/필터/검색/상세)
- 축제달력 페이지
- 게시판 (자유게시판/글쓰기/댓글)
- 로그인 / 회원가입
- 마이페이지 (내정보/비밀번호변경/회원탈퇴)
- AI 챗봇 (오른쪽 하단 팝업)

## 개발 환경
- Node.js 24.12.0
- npm 최신버전

## 브랜치 규칙
- feature/페이지명 → 페이지 개발
- fix/버그명 → 버그 수정
- main → 최종 배포 브랜치
